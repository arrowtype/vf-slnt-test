  • A test can fail in two primary ways:
    • The slant can be wrong ‚Äì either too little too much.
    • The font can be missing required alternates for the given style (Required Variation Alternates, rvrn). In this case, the slant might be correct, but the letterforms are incorrect. For example, the expected result may have a single-story a, while the actual test result has a slanted double-story a.
